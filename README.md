# MEBRU-RSS-5.0
MEBRU RSS Агрегатор 5.0


MEBRU RSS Агрегатор
Это PHP-скрипт, который собирает новости из различных IT и игровых источников в единый RSS-поток.

Описание
Скрипт получает последние статьи с заданных новостных сайтов (таких как Habr, 3DNews, Xakep, iXBT и др.), объединяет их, сортирует по дате публикации и генерирует новый RSS-файл. Это позволяет получать все технологические новости в одном месте, не посещая множество сайтов.

Требования
PHP: версия 8.3 или новее (declare(strict_types=1)).
Расширения:
cURL
libxml
mbstring
Использование

Загрузите файлы скрипта на ваш веб-сервер.
Убедитесь, что у директории со скриптом есть права на запись для создания файла rss.xml.
Запустить скрипт можно двумя способами:
Способ А: Через веб-браузер или Cron
Просто откройте главный PHP-файл в браузере или настройте cron-задание для его периодического вызова.
Скрипт выполнится, получит новости и выведет сформированный RSS-канал прямо в браузер.

Способ Б: Генерация статического файла
При каждом запуске скрипт автоматически сохраняет результат в файл rss.xml в той же папке. Вы можете отдавать этот статический файл пользователям для снижения нагрузки на сервер.


Вы можете легко изменить параметры в начале скрипта:
$max_items: Общее количество новостей в итоговой ленте.
$items_per_source: Сколько новостей брать из каждого отдельного источника.
$sources: Массив с названиями и URL-адресами RSS-лент для агрегации.
